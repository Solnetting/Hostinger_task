Mobile hi-fi screens

Same system, smaller screen

Eight states adapted for mobile. The four-tier routing model is unchanged — only the surfaces adapt. The sidebar becomes a bottom nav. The T4 drawer becomes a bottom sheet. T1 stays persistent above content on every page.

Sidebar nav
→
Bottom navigation — 5 tabs (Home, Websites, Domains, Emails, More). Orange dot migrates from sidebar item to nav tab.
T1 banner
→
Full-width strip directly below topbar — same persistent, non-dismissable behaviour on every page.
T2 cards
→
Full-width cards. Actions (Snooze / Renew) move to a second row below content — cleaner touch targets.
T4 drawer (right)
→
Bottom sheet — pull-only, no badge on Offers tab, same no-push rule.
T3 contextual nudge
→
Inline card on product page, unchanged. Nav tab dot is still the only bridge from Home.
